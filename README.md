# DSA-Practice

This repository contains solutions and practice problems for various Data Structures and Algorithms (DSA) concepts. The topics currently included in this repository are:

## Concepts Covered

- Arrays
- Basic Questions (introductory problems)
- Functions (FunctionQues)
- Striver A2Z (curated problem set from Striver's famous A2Z DSA Sheet)
- KKQues (collection of selected DSA problems)

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Elfinxd/DSA-Practice.git
   ```
2. Explore the topic folders for problems, solutions, and explanations relevant to each DSA concept.
